عند إنهاء أي مهمة في هذا المشروع: 1) شغّل npm run lint. 2) شغّل npm run build إذا كانت التغييرات قد تؤثر على البناء أو المسارات أو الأنواع. 3) تأكد من عدم إدخال ملفات محلية أو حساسة إلى Git؛ .env يجب أن يبقى غير مرفوع و .env.example فقط هو المسموح. 4) حافظ على فصل المنطق بين features وlib/domain وlib/analytics وlib/persistence. 5) إذا كانت المهمة تتعلق بـ Next.js أو سلوك App Router، راجع docs المحلية في node_modules/next/dist/docs بدل الافتراضات القديمة. 6) عند إضافة ميزة جديدة، فضّل تحديث الدليل أو README/PROJECT_STATUS إذا كان لذلك أثر على الاستخدام أو المتابعة.
