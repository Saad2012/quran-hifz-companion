أوامر التطوير الأساسية في هذا المشروع على Windows:
- تثبيت الحزم: npm install
- تشغيل التطوير: npm run dev
- بناء الإنتاج: npm run build
- تشغيل نسخة الإنتاج بعد البناء: npm run start
- فحص eslint: npm run lint
لا يوجد test script مستقل حاليًا.
أوامر مفيدة على Windows/PowerShell:
- عرض الملفات: Get-ChildItem أو dir
- البحث النصي: Select-String أو rg إذا كان متاحًا
- حالة git: git status
- الفروع/السجل: git branch ، git log --oneline --decorate --graph --all
- إضافة/commit/push: git add . ، git commit -m "..." ، git push
أوامر Serena المفيدة خارج Codex عند الحاجة:
- serena init
- serena setup codex
- serena project index <project-path>
- serena project health-check <project-path>
