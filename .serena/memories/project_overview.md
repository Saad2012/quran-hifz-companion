Quran Hifz Companion هو تطبيق ويب عربي RTL لإدارة حفظ القرآن على المدى الطويل. النسخة الحالية local-first وتعمل محليًا بدون backend. الغرض الأساسي: إدارة الحفظ الجديد، المراجعة، الاختبارات، الوقفات، التجويد، الصفحات الضعيفة، التخطيط الأسبوعي، والتحليلات. المكدس التقني: Next.js 16 App Router، React 19، TypeScript strict، Tailwind CSS 4، Radix UI/shadcn-style components، Recharts، Zod، React Hook Form، lucide-react، localStorage. بنية المشروع مقسمة بوضوح: app لمسارات Next والlayouts، components للمكونات العامة، features لواجهات وتدفقات الصفحات، lib/domain لمنطق المجال مثل review-engine، lib/analytics للتحليلات، lib/persistence للتخزين المحلي، types للنماذج typed، data للـ seed والبيانات المساعدة، hooks للهوكس المشتركة، utils للدوال العامة. الحالة العامة للتطبيق تُدار عبر features/app-state/provider.tsx وتُشتق البيانات عبر deriveAppData في lib/domain/app-view.ts.
